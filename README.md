## 개발 환경
- 기간 : 2024/04/24 - 2024/05/27
- OS : Windows 10
- 개발언어 : Java 11.0.18, HTML5, JPS 2.3, Servlet 4
- 서버 : Apache Tomcat/9.0.73
- DBMS : Oracle DB 19.0.0.0.0
- 개발도구 : Eclipse Version 4.26.0
- 인코딩 : UTF-8

## 주요기능
사용자 기능
- 회원가입, 로그인, 예약관리, 리뷰 작성 등

관리자 기능
- 메뉴관리, 예약관리, 매출 통계 확인 등

## 맡은 역할
- 회원가입, 로그인, 자동 로그인
- 메뉴 등록
- 실사용자 리뷰 작성
